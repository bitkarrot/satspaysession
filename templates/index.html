Welcome to {{ title }}!

This micro app delivers a custom Invoice amount Link to the LNBits SatsPayExtension, payable to Bitkarrot.

{{satslink}}

Why do this?

Because we want to be able to specify a BTCPay style lightning invoice in sats with the USD amount (or any other fiat) with a link instead of creating it manually from within the LNBits web interface. Unlike a regular invoice, the satspay server has a nice a count down timer and option for both LN or OnChain payments. We can then share links or add to other applications, such as calendar bookings without extra integration steps.

Alternatively, you can just make a url directly without this form, using the guideline below.

URL Pattern: https://satspaylink.vercel.app/fiat/{currency}/amt/{value}
